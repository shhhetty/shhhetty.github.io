QA Orchestrator — 14-Node Agentic LangGraph Pipeline

Engineered a fully automated, end-to-end keyword QA pipeline as a 14-node LangGraph StateGraph, eliminating 4–5 manual handoffs between QA and Data teams. The pipeline covers keyword staging, product count fetching, sensicality scoring, search-volume retrieval, FAISS-based duplicate detection, assortment scoring with an LLM-as-a-judge, CLP cannibalization checks, page regeneration, and final export — all in ~25 minutes vs. hours across two teams.

Key engineering: conditional guard edges, MemorySaver crash-resumability, human-in-the-loop interrupt gate, FAISS cosine similarity search over 40,000+ CLP keywords (ada-002 embeddings), async DataForSEO SERP polling, and a YAML-driven retailer config system requiring zero code changes to onboard new retailers.

Technologies: Python, LangGraph StateGraph, FAISS, OpenAI API, Anthropic API, DataForSEO API, aiohttp, Pandas
